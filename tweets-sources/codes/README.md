Tweets - Codes
------

This directory contains the Matlab codes to reproduce the figures from the tweets.

The output files of the scripts are generated in directory results/. The directory toolbox/ contains helper functions.

You can find below a representative example of figure generated by some sub-directories. The name of the image file matches the name of the directory.
